``` --- ### 2. 수정된 style.css (레이아웃 수정) 이미지에서 보였던 입력창이 삐져나오는 현상을 막기 위해 `box-sizing` 설정을 강화하고 버튼의 클릭 영역을 확보했습니다. ```css /* 기존 코드의 .input-group 부분을 아래와 같이 수정/보완하세요 */ .input-group { margin-bottom: 25px; /* 항목 간 간격을 넓혔습니다 */ width: 100%; } input[type="text"], select, textarea { width: 100%; padding: 12px; border: 1px solid #ddd; border-radius: 8px; box-sizing: border-box; /* 패딩이 너비에 포함되도록 하여 삐져나감을 방지합니다 */ font-size: 15px; display: block; /* 요소를 블록화하여 아래 요소와 겹치지 않게 합니다 */ } .button-group { margin-top: 30px; position: relative; /* 버튼의 위치를 명확히 합니다 */ z-index: 10; /* 다른 요소보다 위에 오도록 설정합니다 */ } button { width: 100%; padding: 18px; background-color: #4a7c59; /* 이미지의 초록색 느낌으로 변경 */ color: white; border: none; border-radius: 10px; font-size: 18px; font-weight: bold; cursor: pointer; transition: 0.2s; } button:active { transform: scale(0.98); /* 클릭 시 눌리는 효과 추가 */ }
